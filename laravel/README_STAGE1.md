# Stage 1 Scope

Implemented in this stage:

- isolated Laravel 12 app skeleton in `laravel/`
- models for existing database tables
- safe migrations for problem ladders and missing legacy columns
- Filament admin panel provider
- Filament resources for core content, translations, users, tags, media, and ladders
- dashboard widgets
- deployment notes

Not implemented yet:

- public Blade pages
- Livewire practice UX
- bookmarks/progress UI
- ladder public pages
- book/export pages

Those belong to later stages after Stage 1 is installed and verified.
